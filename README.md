#  Análisis de Auditoría de Inventarios

##  Objetivo
Analizar diferencias entre el inventario registrado, ventas y conteo físico para identificar inconsistencias y posibles errores operativos.

---

##  Problema de negocio
Las empresas pueden presentar diferencias entre su inventario teórico y el físico, lo que genera:
- Pérdidas económicas
- Errores en reportes
- Problemas de control interno

---

##  Herramientas utilizadas
- SQL Server
- Python (Pandas)
- Google Colab

---

##  Análisis realizado
- Integración de datos de inventario, ventas y conteo físico
- Cálculo de inventario esperado
- Comparación contra inventario real
- Identificación de diferencias

---

##  Resultados
- 4 de 5 productos presentan diferencias
- El producto **Mouse** tiene la mayor discrepancia

---

##  Insights
- Posibles errores en registro de ventas
- Falta de control en inventarios
- Oportunidad de mejorar procesos de auditoría

---

##  Conclusión
El análisis permite detectar inconsistencias clave que pueden impactar directamente en la operación y toma de decisiones.
